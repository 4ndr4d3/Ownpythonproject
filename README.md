
# Análise de Sentimento em Redes Sociais

Este projeto coleta tweets do Twitter e realiza uma análise de sentimento usando a biblioteca TextBlob.

## Instalação

Use o seguinte comando para instalar as dependências:
```sh
pip install -r requirements.txt

Uso

Execute o script sentiment_analysis.py para coletar tweets e realizar a análise de sentimento.

Estrutura

sentiment_analysis.py: Script principal para coleta e análise de tweets.
data/tweets.csv: Arquivo CSV onde os tweets coletados serão salvos.
images/sentiment_analysis.png: Gráfico gerado pela análise de sentimento.
